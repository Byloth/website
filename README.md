> ### Project setup ###
>
> ```bash
> # install dependencies
> yarn install
>
> # update dependencies
> yarn upgrade
> ```
>
> ---
>
> ### Project build & run ###
>
> ```bash
> # Serve with hot reload at localhost:8080
> $ yarn dev
>
> # Build for production
> $ yarn build
> 
> # Generate static project files
> $ yarn generate
>
> # Launch server
> $ yarn start
>
> # Linting project
> $ yarn lint
> ```
